About the App

This Flask application allows users to select a city and view birding hotspots in that area on a map. It retrieves hotspot data from the eBird API and provides additional information from Wikidata when available. The app also helps users curate hotspots that are missing on Wikidata, which are shown in red on the map.

Written by Tiago Lubiana.

GitHub Repository: https://github.com/lubianat/wikidata_ebird_maps

For those active on iNaturalist, check out the related tool https://inat2wiki.toolforge.org/.
